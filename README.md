# Klasse Signup

A signup page for Klasse, built with React + Tailwind CSS.

## Design Tokens

Colors are mapped from the Klasse Design System in Figma:

- `bg-page` → #FFFEFC (brand/canvas)
- `text-primary` → #300101 (brand/ink)
- `action-primary` → #2216FF (blue/500)

## Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

### Option 1: Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2: GitHub → Vercel
1. Push this to GitHub
2. Go to vercel.com
3. Import the GitHub repo
4. Deploy
